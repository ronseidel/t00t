# t00t

A fart soundboard. Tap to toot.

Live at [t00t.app](https://t00t.app).

## What it is

Exactly what it sounds like. A grid of buttons. Each button plays a fart. That's the whole app.

## Why

Because every portfolio needs one app that isn't trying to prove anything. The 0fam naming scheme demanded it — t00t is right there, staring at you, and refusing to build it felt wrong.

It's also the test case for how far the "single HTML file, no backend, no account" pattern can go. Turns out: pretty far.

## Stack

- Single static HTML file
- Audio files embedded or linked locally
- No backend, no tracking, no account

## Part of 0fam

t00t is part of [0fam](https://0utput.co), a family of small, focused web apps. Every name drops a letter for a zero.
